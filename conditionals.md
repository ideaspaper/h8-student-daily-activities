# Conditionals

Suatu bagian kode program dapat dieksekusi berdasarkan suatu kondisi. Apabila kondisi A terpenuhi, maka kode program bagian pertama dieksekusi. Namun jika kondisi A tidak terpenuhi, maka kode program bagian kedua yang akan dieksekusi. Analogi dalam kehidupan sehari-hari adalah seperti `Jika lapar saya makan. Jika tidak lapar saya tidur.`.

Coba buat algoritma, pseudocode dan kode program untuk permasalahan di bawah:

1. Terdapat sebuah alat pemantau kecepatan mobil. Apabila kecepatan mobil di atas 80 km/jam, maka tampilkan `alarm berbunyi`. Jika kecepatan mobil 80 km/jam ke bawah, maka tampilkan `alarm tidak berbunyi`.
1. Buat sebuah program untuk mengubah angka dalam bentuk string (`satu`, `dua`, `tiga`, ...), menjadi bentuk integer (`1`, `2`, `3`, ...).
1. Diberikan 3 buah titik, yaitu A, B dan C. Masing-masing titik memiliki suatu nilai koordinat. Apabila jarak dari A dan B lebih kecil dari A dan C, tampilkan `Menuju ke B`. Apabila jarak dari A dan C lebih kecil dari jarak dari A dan B, tampilkan `Menuju ke C`. Apabila jaraknya sama, tampilkan `Diam di tempat`.
